# Message Organizer V4

교회 메시지 개요를 입력하면

- A4 문서 형식으로 자동 정리
- PDF 저장
- 모바일(iPhone/iPad) 지원
- Mac 지원
- Vercel 배포 가능

## 파일 구성

- index.html
- style.css
- parser.js
- script.js
- README.md

## 배포

GitHub 저장소에 업로드 후

Vercel → Add New Project

로 배포 가능

## 주요 기능

- 본문 / 서론 / 본론 / 결론 자동 인식
- 제목 자동 정렬
- 대주제 / 소주제 자동 정리
- `(1) (2) (3)` 세부소주제 자동 정렬
- PDF 저장
